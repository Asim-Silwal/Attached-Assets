# Website Strategy

## Primary Purpose
The website exists to build trust and generate discovery calls.
It should act as a conversion-focused sales presence for Vertex Digital, not just a portfolio showcase.

## Main Website Goals
- Increase trust
- Generate discovery calls
- Show professionalism
- Explain the process clearly
- Position Vertex Digital as an expert in home service businesses
- Build credibility for international clients

## Core UX Questions
Every page should answer these quickly:
- Who are we?
- Who do we help?
- Why should someone trust us?
- What problem do we solve?
- What should they do next?

## Current Website Structure
The current implementation is a single-page experience with sections that cover:
- Hero messaging
- Credibility
- Industries served
- Services
- Portfolio or case-study style proof
- Why Vertex Digital is different
- Process
- Testimonials
- FAQ
- Contact CTA

This structure is appropriate for the current stage, provided the content remains sharp and the navigation stays effortless.

## Strategic Direction
The site should evolve by strengthening clarity, proof, and trust before adding more visual complexity.

Recommended priorities:
- Improve CTA clarity and form experience
- Strengthen proof and trust signals
- Tighten messaging so it feels less generic and more specialized
- Improve technical and performance quality
- Keep the navigation simple and predictable

## Conversion Principles
- One primary action per page whenever possible
- Make the next step obvious
- Reduce form friction
- Keep the process visible and easy to understand
- Make trust evidence easy to scan

## Long-Term Website Vision
Over time, the site may grow from a single landing page into a deeper conversion system with dedicated pages for:
- Services
- Industries
- Case studies
- SEO landing pages
- Process and FAQ content

That expansion should only happen when the core message is already strong.
