# Vertex Digital Docs

This directory is the working documentation system for Vertex Digital.
It explains the business, brand, strategy, design direction, code architecture, SEO, security, roadmap, and AI working rules that should govern future development.

## How to Use These Documents
- Start with [00_PROJECT_OVERVIEW.md](00_PROJECT_OVERVIEW.md) to understand the business and the current product direction.
- Use [01_BRAND_GUIDELINES.md](01_BRAND_GUIDELINES.md), [03_TARGET_AUDIENCE.md](03_TARGET_AUDIENCE.md), and [06_CONTENT_GUIDELINES.md](06_CONTENT_GUIDELINES.md) before making copy or UI decisions.
- Use [05_DESIGN_SYSTEM.md](05_DESIGN_SYSTEM.md) and [07_DEVELOPMENT_GUIDELINES.md](07_DEVELOPMENT_GUIDELINES.md) before changing the interface or adding new components.
- Use [08_CODE_ARCHITECTURE.md](08_CODE_ARCHITECTURE.md), [09_SEO_STRATEGY.md](09_SEO_STRATEGY.md), and [10_SECURITY_CHECKLIST.md](10_SECURITY_CHECKLIST.md) before technical work.
- Use [11_ROADMAP.md](11_ROADMAP.md) and [12_TODO.md](12_TODO.md) to understand priorities.
- Use [14_AI_INSTRUCTIONS.md](14_AI_INSTRUCTIONS.md) when asking AI tools or agents to contribute to this project.

## Current Scope
Vertex Digital is a U.S.-focused growth partner for home service businesses.
The current website is a premium, conversion-focused marketing site built with a React/Vite frontend and a small Express API service.

## Documentation Principles
- Do not invent business claims or client results.
- Keep the brand premium, calm, and trustworthy.
- Favor clarity, modularity, and accessibility.
- Use documentation to keep future implementation decisions consistent.
