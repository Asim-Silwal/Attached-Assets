# Design System

## Design Goals
The interface should feel:
- Clean
- Spacious
- Consistent
- Easy to scan
- Minimal
- Premium

The design should support a premium digital agency feel without becoming flashy or overly futuristic.

## Current Visual Direction
The current site already uses a dark, premium aesthetic with strong contrast, purple accent color usage, polished cards, and motion-led section transitions.
That direction is compatible with the brand if it remains restrained and readable.

## Typography
Typography should support:
- Clear hierarchy
- High readability
- Calm premium tone
- Strong section scanning

Guidelines:
- Keep headings bold and confident
- Keep body copy readable and concise
- Avoid decorative typography that reduces clarity

## Color Principles
- Use a restrained palette
- Keep accent colors deliberate and consistent
- Maintain strong contrast for readability
- Avoid overusing bright accent colors as decoration

## Layout Principles
- Use whitespace intentionally
- Prefer simple grid structures
- Keep cards and sections visually balanced
- Make calls to action obvious without overwhelming the page

## Motion Principles
Motion should:
- Reinforce hierarchy
- Support delight in small doses
- Help users orient themselves
- Stay subtle enough to preserve calmness

Avoid:
- Excessive parallax
- Gimmicky transitions
- Constant motion without functional purpose

## Component System Expectations
Reusable components should be preferred for:
- Buttons
- Sections
- Cards
- Navigation
- Form elements
- Status and feedback states

## Design Quality Checks
Before approving a UI change, confirm that it:
- Matches the brand personality
- Improves clarity
- Works on mobile and desktop
- Preserves accessibility
- Does not add unnecessary visual noise
